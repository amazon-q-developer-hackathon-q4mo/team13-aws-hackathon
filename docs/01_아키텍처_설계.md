# LiveInsight 아키텍처 설계

## 전체 아키텍처

### 1. 이벤트 수집 파이프라인
```
정적 스크립트 (Vanilla JS) 
    ↓
API Gateway 
    ↓
Lambda Function 
    ↓
DynamoDB
```

### 2. 데이터 조회 및 대시보드
```
프론트엔드 대시보드 
    ↓
Django REST API 
    ↓
DynamoDB 조회
```

## AWS 서비스 구성

### API Gateway
- 이벤트 수집 엔드포인트
- CORS 설정
- 요청 검증 및 제한

### Lambda Functions
- 이벤트 처리 로직
- 세션 관리
- 데이터 변환 및 저장

### DynamoDB
- 이벤트 데이터 저장
- 세션 정보 관리
- 실시간 조회 최적화

### Django 서버
- 통계 API 제공
- 실시간 데이터 조회
- 대시보드 백엔드

## 데이터 플로우

1. **이벤트 수집**: 웹사이트 → API Gateway → Lambda → DynamoDB
2. **실시간 조회**: 대시보드 → Django → DynamoDB
3. **통계 생성**: 배치 처리 → 집계 데이터 생성
